🍔 Lieferspatz – Food Delivery Platform

Built a full-stack food delivery platform backend inspired by systems like Uber Eats and Lieferando.

The platform supports customers, restaurants, and administrators with real-time order updates and a modular API architecture.

Key features:

• Customer ordering system
• Restaurant dashboard for order management
• Admin monitoring tools
• JWT authentication & role-based access
• Wallet and voucher payment system
• Real-time order updates using WebSockets
• Modular FastAPI backend architecture

Technologies used:

Python, FastAPI, SQLAlchemy, REST APIs, WebSockets, SQLite

This project demonstrates:

API architecture

backend system design

authentication systems

database modeling

real-time communication
